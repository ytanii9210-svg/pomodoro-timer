# 🍅 Pomodoro Timer

シンプルなポモドーロタイマーです。集中・休憩・長休憩の3モードに対応しています。

## 使い方

`index.html` をブラウザで開くだけで動作します。インストール不要です。

## 機能

- **集中** (25分) / **休憩** (5分) / **長休憩** (15分) の切り替え
- タイマー完了時の通知
- 今日の完了数とストリーク表示
- スペースキーでスタート/ポーズ
- ライトモード / ダークモード 自動対応

## キーボードショートカット

| キー | 動作 |
|------|------|
| `Space` | スタート / ポーズ |
